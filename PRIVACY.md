# Cairn Privacy

Cairn does not upload data, send telemetry, use analytics, or require an
account. It runs locally.

Memory is disabled by default. When enabled, Cairn may read local AI transcript
files to build a project-scoped Obsidian vault. By default, that vault stores
only changed file paths, operation labels, and timestamps; it does not copy
full prompt text. You can disable memory at any time and delete the generated
vault without affecting your source project or original transcript files.
