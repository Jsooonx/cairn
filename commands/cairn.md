description: Start Cairn project setup or manage its private local memory.
---

Invoke the bundled `cairn` skill. If the current project has no `.cairn.json`,
begin with the privacy onboarding before writing any file. Present Off,
On-demand (recommended), and Workflow auto memory modes with their local-data
disclaimer. Treat `/cairn sync`, `/cairn memory enable`, `/cairn memory disable`,
and `/cairn memory remove` as explicit memory-management requests.
